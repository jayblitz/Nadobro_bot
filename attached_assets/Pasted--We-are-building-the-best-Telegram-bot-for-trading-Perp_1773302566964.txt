### How We Use It in Alpha Agent Mode

We call the LLM in two places only:

1. **Opportunity Scanner** (every 5 minutes)  
2. **HOWL nightly optimization** (auto-tuning)

Here’s the tiny integration code (add this to your existing files):

#### `src/nadobro/services/llm_agent.py` (new file)

```python
import os
from openai import AsyncOpenAI  # or xAI client if you prefer

client = AsyncOpenAI(
    api_key=os.getenv("XAI_API_KEY"),
    base_url="https://api.x.ai/v1"   # Grok endpoint
)

async def llm_decide_opportunity(market_data: dict) -> dict:
    prompt = f"""
    You are Nadobro's Alpha Agent. Analyze this market snapshot and decide the best action.
    Return ONLY valid JSON.

    Market: {market_data}

    Possible actions: grid_long, grid_short, delta_neutral, volume_farm, skip
    """
    
    response = await client.chat.completions.create(
        model="grok-3-mini",
        messages=[{"role": "user", "content": prompt}],
        temperature=0.3,
        max_tokens=300
    )
    
    return eval(response.choices[0].message.content)  # safe JSON parse in production
```

Then call it inside your Alpha Agent:

```python
# In alpha_agent.py
opportunities = await self._scan_opportunities()
for opp in opportunities:
    decision = await llm_decide_opportunity(opp)
    if decision["action"] != "skip":
        await self._execute_slot(decision)
```
