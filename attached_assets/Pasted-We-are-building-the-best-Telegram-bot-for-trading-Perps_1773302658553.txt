### 1. AI Failure Recovery & Debugging
Added exponential backoff retries for API calls and structured logging with error categorization. This ensures the agent recovers from transient issues (e.g., network blips) without crashing.

**Update to `alpha_agent.py` (in `_execute_slot` method):**
```python
import logging
import backoff  # pip install backoff

logger = logging.getLogger("nadobro_agent")

@backoff.on_exception(backoff.expo, Exception, max_tries=3)
async def _execute_slot(self, slot):
    try:
        proposed = slot["size"] * slot["price"]
        allowed, reason, safe_size = await can_execute_order(self.user_id, proposed, slot["strategy"])
        
        if not allowed:
            await self._send_alert(f"🛑 Agent stopped: {reason}")
            await emergency_flatten(self.user_id)
            return False
        
        slot["size"] = safe_size / slot["price"]
        success = await self.client.place_order(**slot)
        
        if success:
            await update_exposure(self.user_id, safe_size)
        else:
            raise ValueError("Order execution failed")
        return success
    except Exception as e:
        error_type = "network" if "connect" in str(e) else "margin" if "margin" in str(e) else "unknown"
        logger.error(f"[DEBUG] User {self.user_id} | Error Type: {error_type} | Details: {str(e)} | Slot: {slot}")
        await self._send_alert(f"⚠️ Recovery in progress: {str(e)}")
        await emergency_flatten(self.user_id)
        return False
```

### 3. Data Ingestion & Processing
Added preprocessing to validate and clean market data before feeding to LLM, preventing garbage-in-garbage-out issues. Used simple ETL-like filtering.

**Update to `llm_agent.py` (new function before LLM calls):**
```python
async def ingest_market_data(raw_data: dict) -> dict:
    # Validate and clean (ETL step)
    cleaned = {k: v for k, v in raw_data.items() if v is not None and (isinstance(v, (int, float, str)) or isinstance(v, dict))}
    if "price" not in cleaned or cleaned["price"] <= 0:
        raise ValueError("Invalid market data - skipping")
    return cleaned

# In llm_analyze_opportunity and llm_optimize_howl:
```

```python
schema = {
```
