# Scripts

## setup_secrets.py

Same logic as the root-level **run_setup_secrets.py**. Prefer running from project root:

```bash
python run_setup_secrets.py
```

That way `.env` is created next to `main.py` no matter where the `scripts/` folder lives. See [docs/SECRETS_SETUP.md](../docs/SECRETS_SETUP.md) for full details.
